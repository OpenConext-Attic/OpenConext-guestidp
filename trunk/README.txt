==============================================================================
===                             COIN-GUESTIDP                              ===
==============================================================================

The GuestIdP is an open-source on Simplesamlphp based IdP which enables
authentication with SURFguest, facebook, twitter, hyves, yahoo, google and
virtualy any openid provider.
